Q1. What is the Difference between react and react native? Which one is library or framework?

React Native is a React JS-based framework to design mobile apps. Moreover, with reusable components, you can develop native mobile applications.React is a JavaScript library of reusable components designed to create skeletons of the apps with reusable components. React is an open-source JS library for building the UIs for web applications.

Q2. Which one is library or framework?

React is an open source JavaScript framework, that isn't actually a framework.
React Native is a JavaScript framework for writing real, natively rendering mobile applications for iOS and Android.

What is the package name you are using for routing?

React Router is a standard library system built on the React and used to create routing in the React application using React Router Package.

Q3. What is lazy loading in react ?

lazy loading means that a component or a part of code must get loaded when it is required. It is also referred to as code splitting and data fetching .
